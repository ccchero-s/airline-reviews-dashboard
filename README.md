# Airline Reviews Dashboard

An interactive dashboard built with Python, Dash, Plotly, and Pandas to analyse airline customer reviews, recommendation rates, and satisfaction trends across different airlines, travel classes, and time periods.

## Project Overview
This project transforms airline review data into an interactive dashboard that allows users to explore customer feedback and compare airline performance. The dashboard helps users identify patterns in recommendation rates, review scores, and passenger experience by applying dynamic filters.

## Features
- Filter reviews by start year, end year, start month, end month, and travel class
- Select specific airlines for focused analysis
- View recommendation rate through an interactive gauge chart
- Explore customer satisfaction metrics in a structured table
- Read detailed customer review text based on selected filters
- Analyse review trends across time and passenger segments

## Tools & Technologies
- Python
- Dash
- Plotly
- Pandas

## Dataset
The dataset contains airline customer reviews, including:
- Airline name
- Review date
- Travel class
- Recommendation result
- Customer ratings and review details
