# Airline Reviews Dashboard

## Project Overview
This project presents an interactive dashboard designed to analyse airline customer reviews and passenger satisfaction trends. The dashboard transforms raw review data into clear, visual insights that allow users to explore airline performance across different time periods, travel classes, and traveller types.

By combining data analysis and interactive visualisation, the dashboard enables users to better understand how customer experiences vary across airlines and identify key patterns in passenger feedback.

## Business Objective
Airlines receive large volumes of customer feedback, but extracting meaningful insights from raw review data can be challenging.

The objective of this project is to develop a data-driven dashboard that helps users:
- Analyse passenger satisfaction across different airlines
- Identify patterns in recommendation rates
- Compare service performance across travel classes
- Explore customer feedback trends over time

This dashboard demonstrates how data visualisation tools can support more effective exploration of customer experience data.

## Tools & Technologies

## Key Dashboard Features
### Interactive Time Filters
Users can select:
- Start year and end year
- Start month and end month

This allows exploration of customer satisfaction trends across different time periods.

### Travel Class Filtering
Passenger reviews can be analysed by travel class, enabling comparisons between different service levels.

### Airline Selection
Users can select a specific airline to access its official website and review relevant information.

### Recommendation Rate Indicator
A gauge chart displays the percentage of passengers who would recommend the airline based on filtered data.

### Customer Satisfaction Table
A sortable table presents detailed service ratings, including:
- Seat Comfort
- Staff Service
- Food & Beverages
- Inflight Entertainment
- Value for Money
- Overall Rating

### Customer Review Exploration
The dashboard also displays individual customer reviews, providing qualitative insights into passenger experiences.

### Service Performance Heatmap
A heatmap visualisation highlights the average rating of airlines across different traveller types, helping users quickly identify patterns in service performance.
